# [Forcastly](https://demos.creative-tim.com/light-bootstrap-dashboard-react/#/?ref=lbdr-readme)



![version](https://img.shields.io/badge/version-1.0.0-blue.svg) ![license](https://img.shields.io/badge/license-MIT-blue.svg)

**[Forcastly](https://inventory-forecastly.herokuapp.com/)** is an inventory forecasting tool designed to be useful for any company that wants to know how much inventory one needs on hand. It is built with an algorithym design to take as much input as needed from the user to accurately display how many units will be necesary to have on hand.

The product represents an easy tool that can help you jump start your inventory on hand for the future. We have created it thinking about things you actually experience as a business. Everything is designed to fit any size company from handling 10 sku's to 1,000,000 sku's. As you will be able to see, the dashboard is utilize to view the most sold sku's by dollar amount and by quantity.

## Reporting Issues
We use GitHub Issues as the official bug tracker for the Light Bootstrap Dashboard React. Here are some advices for our users that want to report an issue:

1. Make sure that you are using the latest version of the Light Bootstrap Dashboard React. Check the CHANGELOG from your dashboard on our [website](https://www.creative-tim.com/?ref=lbdr-readme).
2. Providing us reproducible steps for the issue will shorten the time it takes for it to be fixed.
3. Some issues may be browser specific, so specifying in what browser you encountered the issue might help.
